MarkNET - NUMBERS

MarkNET is a neural network implementation for handwritten digit recognition, developed using C programming language with a Node.js backend API. This project demonstrates multi-layer perceptron architecture with ReLU activation functions, trained on the MNIST dataset to classify digits 0-9 with high accuracy.
